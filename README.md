# 基于Hexo的个人博客源文件

Hexo的使用：<https://github.com/hexojs/hexo>
查看网站内容可访问<https://zilchme.github.io/>

## Installation

### 安装 nodejs、npm

```sh
sudo apt update
sudo apt install nodejs npm
```

更新 nodejs：

```sh
sudo npm install n -g
sudo n stable
```

### 安装 hexo

```sh
sudo npm install webpack -g
sudo npm install -g hexo-cli
```

安装项目 npm 依赖：

```sh
sudo npm install
```

### 本地部署

```sh
hexo clean
hexo g
hexo s
```
